Admin Pages (Users, Forms, Fields)

Same page structure, same components
Admin pages follow the identical layout pattern: sidebar + top bar + content. Key changes:
  • User Management — Table with avatar, name, email, role badge, status. Invite modal uses standard modal pattern. Bulk actions via multi-select.
  • Forms & Fields — Card grid for form templates (each card shows template name, type badge, field count, last modified). Click to open form builder.
  • Form Builder — Drag-and-drop field layout with live preview. Uses standard Input, Select, TextArea components in the preview.
  • Field Lookups — Table with field name, type, validation rules. Edit inline or via modal.
  • Workspaces — Card-based workspace overview with member avatars and quick actions.
Key principle: Admin pages are not special. They use the exact same components and layout as user-facing pages. No separate "admin theme."
