Typography Scale

Montserrat for headings, Inter for body — already defined, now enforced everywhere
Page Title (H1) — 28px Montserrat Bold
Used for: main page headings (Dashboard, Requests, etc.)
Section Heading (H2) — 20px Montserrat Bold
Used for: card titles, modal headers, section breaks
Subsection (H3) — 16px Inter Semibold
Used for: table headers, form section labels
Body text — 14px Inter Regular. This is the standard reading size used for table content, form descriptions, and general paragraphs.
Small text — 12px Inter Regular. Used for helper text, timestamps, secondary info.
Overline / Label — 10px Inter Semibold Uppercase
Used for: stat card labels, table column headers, section labels
