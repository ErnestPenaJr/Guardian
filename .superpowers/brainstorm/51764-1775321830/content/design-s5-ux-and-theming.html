Implementation approach: CSS custom properties (variables) on :root for light mode, [data-theme="dark"] for dark mode. Tailwind's dark: variant for utility classes. Theme preference stored in localStorage with prefers-color-scheme media query fallback for "System" mode. Transition: background-color 0.2s, color 0.2s for smooth switching.
